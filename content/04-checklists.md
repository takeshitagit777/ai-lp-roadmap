# チェックリスト

## LP品質チェック

- [ ] ファーストビューで何のサービスか分かる
- [ ] CTAボタンがすぐ見つかる
- [ ] スマホで文字が読める
- [ ] 画像が不自然に切れていない
- [ ] 悩み訴求が具体的
- [ ] 料金または相談導線が分かる
- [ ] FAQで不安を減らしている
- [ ] フッターに連絡先や事業情報がある
- [ ] npm run build が成功する

## GitHubチェック

- [ ] リポジトリ名が分かりやすい
- [ ] READMEがある
- [ ] node_modules が .gitignore に入っている
- [ ] main ブランチにpush済み
- [ ] package.json の scripts が正しい

## Vercelチェック

- [ ] Framework Preset が適切
- [ ] Build Command が `npm run build`
- [ ] Output Directory が Viteなら `dist`
- [ ] 静的HTMLなら `public` または空欄を確認
- [ ] 画像パスが本番で崩れていない
- [ ] 公開URLを開いて確認した
- [ ] スマホ表示も確認した

## ポートフォリオ掲載チェック

- [ ] liveUrlが正しい
- [ ] スクリーンショットを取得した
- [ ] カード画像が素材写真ではなく実画面になっている
- [ ] タイトルと説明文が自然
- [ ] GitHub URLが正しい
- [ ] npm run build が成功
- [ ] 本番ポートフォリオで表示確認

## 販売前チェック

- [ ] 商品名
- [ ] 価格
- [ ] 内容物
- [ ] 提供方法
- [ ] 返金条件
- [ ] 特定商取引法に基づく表記
- [ ] プライバシーポリシー
- [ ] 問い合わせ先
- [ ] 成果保証ではない旨の注意書き
- [ ] 決済完了後の案内文
