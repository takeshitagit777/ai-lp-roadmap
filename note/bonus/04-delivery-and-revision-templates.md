# 納品・修正テンプレート

## 初稿提出

```text
お世話になっております。
LPの初稿ができましたので共有いたします。

公開確認URL:
{URL}

今回、特に意識した点は以下です。
- {意識した点1}
- {意識した点2}
- {意識した点3}

スマートフォンでも見やすいように、問い合わせ導線を目立たせています。
ご確認いただき、修正希望があればまとめてお送りください。
```

## 修正依頼への返信

```text
ご確認ありがとうございます。
いただいた修正点、確認しました。

対応内容:
- {修正1}
- {修正2}
- {修正3}

修正完了後、再度URLを共有します。
追加で大きな構成変更が出る場合は、作業量を見て事前にご相談します。
```

## 納品完了

```text
LPの修正が完了しました。

公開URL:
{URL}

納品内容:
- 1ページLP
- PC / スマホ対応
- CTA設置
- 公開設定

今後、文章変更や画像差し替えなど軽微な修正が必要になった場合はご相談ください。
このたびはご依頼ありがとうございました。
```

## 実績掲載のお願い

```text
今回制作したLPを、私の制作実績としてポートフォリオに掲載してもよろしいでしょうか。

掲載内容は以下の範囲です。
- サイト名
- 公開URL
- ファーストビューのスクリーンショット
- 簡単な制作説明

掲載NGの情報があれば、その部分は伏せて対応します。
```
