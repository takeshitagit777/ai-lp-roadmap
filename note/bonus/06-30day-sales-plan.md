# 30日営業ロードマップ

このロードマップは、LPを作って終わりにしないための実行表です。

## Week 1: 見せられる実績を作る

目標：
公開URLつきのデモLPを1本作る。

やること：
- 業種を1つ選ぶ
- 参考画像を1枚決める
- AIに構成を出させる
- LPを作る
- GitHub / Vercelで公開する
- スクリーンショットを撮る
- ポートフォリオに載せる

## Week 2: 営業材料を整える

目標：
営業で送れる材料を作る。

やること：
- Before / Afterの説明文を作る
- 制作意図を3行で書く
- 営業文を3パターン作る
- 業種別に改善ポイントを整理する
- 提案用スクリーンショットを用意する

## Week 3: 30件リストアップする

目標：
営業先を30件集める。

探し方：
- Googleマップ
- Instagram
- 地域名 + 業種
- 古いサイトの事業者
- 予約導線が弱い店舗
- スマホで見づらいサイト

見るポイント：
- 予約ボタンが見つけやすいか
- 料金が分かりやすいか
- スマホで読みやすいか
- 写真が古くないか
- 問い合わせ前の不安が消えているか

## Week 4: 送って改善する

目標：
10件以上送って、返信から改善する。

やること：
- 1日2件から3件送る
- 返信が来た文面を保存する
- 反応がない文面は短くする
- 相手別の改善案を1つ入れる
- 無料ラフを作りすぎない

## 毎日見る数字

- リスト追加数
- 送信数
- 返信数
- 無料ラフ作成数
- 商談数
- 受注数

最初は返信が少なくても普通です。
大事なのは、相手のサイトを見たうえで送ることと、送った後に文面を直すことです。
