# 購入者向け：最初に読むページ

この教材は「AIでLPを作る方法」だけではなく、公開、実績化、営業までを1本の流れにした実行キットです。

最初から全部を完璧にやろうとすると止まりやすいので、まずは次の順番で進めてください。

1. 参考画像を1枚決める
2. AIにLP構成を分解させる
3. Codex / ChatGPTでLPを実装する
4. GitHubへアップする
5. Vercelで公開する
6. スクリーンショットを撮る
7. ポートフォリオに載せる
8. 営業先を30件集める
9. 改善案つきで10件送る
10. 反応を見て文面を直す

大事なのは、1本目から売ろうとしすぎないことです。

まずは「公開URLがある」「見せられる実績がある」「営業文が送れる」状態を作ります。

## 最短ルート

時間がない人は、この順番だけ守ってください。

- Day 1: 参考画像を決めて、AIに構成を出させる
- Day 2: LPを実装する
- Day 3: スマホ表示と文章を直す
- Day 4: GitHub / Vercelで公開する
- Day 5: ポートフォリオに載せる
- Day 6: 営業リストを30件作る
- Day 7: 10件送って反応を見る

## 失敗しやすいポイント

- 画像の見た目だけ真似して、問い合わせ導線を作らない
- 公開前にずっとデザイン修正を続ける
- ポートフォリオに載せず、営業で見せられるものがない
- 営業文が自分語りになっている
- 相手のサイトを見ずにテンプレ文だけ送っている

AIは作業を速くできますが、相手のページを見て「どこを直すと成果につながりそうか」を考えるのは人間の仕事です。
